API Reference
=============

This section of the documentation provides a detailed reference for the Fakernaija library's API. The API is organized into several modules, each responsible for a different aspect of the library's functionality.

The API is divided into three main categories:

- **Mixins**: Classes that provide reusable functionality across various data types.
- **Commands**: Command-line interface (CLI) commands for generating and managing fake data.
- **Providers**: Classes responsible for supplying fake data based on specific categories.
